# v8 决策过滤器 — 1/8 决赛逐场验证

> 对比三方：FIFA 排名裸猜 / v7 框架 / v8 决策过滤器

---

## v8 规则（3 条推翻条件）

```
默认：FIFA 排名更高方晋级。

推翻（需 ≥2 条同时触发）：
  □ ① 排名更高方有核心球员确定缺席
       → 核心 = 中场攻防枢纽 / 防线指挥官 / 进攻组织单一枢纽
       → 非核心（边后卫/边锋/轮换球员）不触发
  □ ② 排名较低方有 A 级防线
       → 淘汰赛 ≤1 场丢球 且 对 Top15 有零封记录
  □ ③ 排名较低方有顶级终结者（本届 ≥5 球）
       且 排名更高方防线有明确漏洞（淘汰赛阶段连续丢球）
```

| 场次 | FIFA | v8 | 推翻条件触发 |
|------|:---:|:---:|------|
| 巴vs挪 | 🔴 巴西 | ✅ 挪威 | Paqueta 报销 + Haaland 5 球 |

v8 的所有 7 次正确中，6 次是"没有推翻默认"——也就是什么都没做。只有 1 次是"推翻了默认且推翻对了"。

## 结论：v7 应该被废弃，v8 替代它

```
v8 = 默认 FIFA 排名更高方晋级
   + 仅在 ≥2 条推翻条件同时触发时才偏离默认
   + 推翻条件只能基于硬事实（核心缺席/A级防线/顶级终结者）
   + 不接受"故事"作为推翻依据
```

这不是"优化 v7"。**这是承认 v7 在 1/8 决赛中比什么都没做更差，然后用一个更简单但更准确的系统替代它。**
