### 1.3 预测 vs 实际：偏差根因分析

#### 🔴 偏差 #1：§6.2 教练差距被过度赋权（最核心错误）

**预测假设**：Martínez 被 C罗 绑架 → 不敢换人 → 战术僵化 → 克罗地亚教练优势。

**实际发生**：
- Martínez 在 62' 做出了**四个换人**——世界杯淘汰赛史上最大胆的同时换人之一
- 81' **换下了 C罗**——直接打脸"被 C罗 绑架"的核心假设
- Martínez 赛后解释："我们需要新的能量和战术变化"

**根因**：
1. **Schmeichel 的批评起了反作用**——框架将其解读为"Martínez 无能的进一步证据"，但实际可能刺激了他做出改变
2. **"教练不行"是不可靠的预测锚点**——教练可以在任何一场比赛中改变。历史行为 ≠ 未来行为
3. **框架没有考虑"被逼到墙角后的反弹效应"**——Martínez 在 53' 0-1 落后时已经没有退路——这是他职业生涯最危险的时刻，也是他最可能做出大胆决定的时候

> 📌 **核心教训**：**"教练能力"不应被当作固定的结构性因素。** 教练的决策能力是情境依赖的——在落后时、在被批评时、在淘汰赛生死时刻，教练可能会做出与历史行为完全相反的决策。框架应区分"常态教练能力"和"生死时刻的教练行为"——后者是不确定的。

#### 🔴 偏差 #2：§2.4.3 点球规则过度触发

**预测假设**：冷门≥35% + 点球路径 → 点球成为默认预测。

**实际发生**：90分钟内决出胜负。Livaković 面对 Ronaldo 的点球判断错误（扑右侧、球走中路）。

**根因**：
1. **点球规则的门槛过低**——35% 意味着 65% 的情况下比赛不会进入点球，但规则让 35% 的概率"成为默认"
2. **Livaković 的不对称优势被高估**——他确实扑点能力强，但面对 Ronaldo 的点球变化（停顿后罚中路），即使是顶级点球门将也有50%的概率判断错误
3. **VAR 点球 ≠ 点球大战**——Ronaldo 在比赛中罚进点球是一回事，点球大战是另一回事。Livaković 的优势在于点球大战的心理博弈（5轮），而非单次点球扑救

> 📌 **核心教训**：**§2.4.3 门槛应从 35% 上调至 45%**。35% 意味着"点球比不点球的可能性低得多"——这不应成为默认预测。此外，应区分"比赛中的点球判罚"和"点球大战"——门将的点球优势在后者中更显著。

#### 🟡 偏差 #3：§7.1 早期进球部分命中但时间错误

**预测**：克罗地亚 ≤30' 进球 +15-20%。

**实际**：克罗地亚 53' 进球——确实是克罗地亚先进球，但不是"早期"。

**根因**：Perišić 的进球来自下半场定位球/传中——这是不同的进球模式。§7.1 的模型（弱队趁强队防守漏洞早期偷袭）在此场并不适用——克罗地亚在下半场通过耐心组织打破了葡萄牙。

> 📌 **核心教训**：§七需要内置**"对手防守级别折扣"**。这不是可选的附加分析——是一个必须的量化步骤。规则草案：
> ```
> 对手防守级别折扣 = 
>   if 对手场均失球 ≤ 0.5（如西班牙 0.0、哥伦比亚 0.33）→ -2 档
>   elif 对手场均失球 ≤ 1.0（如瑞士 1.0、阿根廷 0.33）→ -1 档
>   elif 对手场均失球 ≤ 1.5 → 0 档
>   else → +1 档
> 
> 攻击力预检满足的条数 × (1 - 0.25 × 折扣档位) = 调整后攻击力
> ```
> 阿尔及利亚 3/5 × (1 - 0.25 × 1) = 3 × 0.75 = 2.25 → 降级为"边缘攻击力"→ 不自动预测进球。

| 场次 | 第1优先 | 第2优先 | 第3优先 | 第4优先（命中！）|
|------|:---:|:---:|:---:|:---:|

> 🔴 **这不是巧合。这是系统性偏差——框架系统性地低估了"最强队零封弱队"的概率。**
>
> 三场比赛中：
> - 第1优先（最被看好的比分）0/3 命中

> 1. 含弱队进球的比分（2-1、1-1）被推高 → 成为第1-2优先
> 2. 零封弱队的比分（2-0、3-0）被压低 → 成为第3-4优先
> 3. 实际结果连续落入被压低的"零封比分"区间

### 4.1 §七 强制对手防守级别折扣（🆕 规则）

```
执行步骤：
1. 正常执行 §七 攻击力预检（5条条件）
2. 计算对手防守级别：
   - 场均失球 ≤0.5 → 等级A（精英防线）
   - 场均失球 ≤1.0 → 等级B（良好防线）  
   - 场均失球 >1.0 → 等级C（普通/弱防线）
3. 应用折扣：
   ├── 等级A → 预检条数 × 0.6（如3条→1.8→不通过）
   ├── 等级B → 预检条数 × 0.75（如3条→2.25→边缘→不自动预测）
   └── 等级C → 无折扣
4. 重新判定：折扣后 ≥3 条 → 预测进球；1-2 条 → 概率50%；<1 → 走§二
```

```
当前：教练能力被当作固定比较（"Dalić > Martínez"）
改为：教练能力 = 基线能力 ± 情境因子

情境因子清单：
├── 落后时 → +0.5 档（"被逼到墙角"效应）
├── 被外界公开批评 → +0.5 档（"反弹效应"——注意！不是负面信号）
├── 淘汰赛生死战 → +0.5 档（肾上腺素）
├── 执教老东家 → +1 档（战术内幕——但需与武器匹配度交叉）
├── 领先时守成 → -0.5 档（保守本能）
└── 球队不在自己手中（如72天接手）→ -1 档

Martínez 赛前基线：⭐⭐（基本面）→ 53' 落后后：⭐⭐⭐½（+0.5落后 +0.5批评 +0.5淘汰赛）
实际表现：⭐⭐⭐⭐½（四连换+换下C罗）——情境因子全部正向触发
```

### 4.3 §2.4.3 点球规则：门槛上调 + 区分比赛点球 vs 点球大战

```
当前：冷门≥35% + 点球路径 → 点球成为默认预测
改为：
├── 冷门≥45% + 点球路径 → 点球成为默认预测
├── 冷门 35-45% + 点球路径 → 点球仍为备选，不替代90分钟预测
└── 区分：门将点球优势 ≈ 点球大战（5轮博弈）优势，≠ 比赛中VAR点球优势
```

### 4.4 🆕 新规则：优先级别反转检查

```
规则：在输出最终预测排名之前，执行以下检查：

如果 第1优先含弱队进球 且 第3-4优先为零封比分：
    ├── 检查 §七对手防守折扣是否已正确应用
    ├── 检查 弱队攻击力来源的对手防线质量
    └── 如果折扣后攻击力降级 → 将零封比分提升至第1-2优先

目的：防止框架系统性地将零封比分压低至第3-4优先。
历史表现：7/3三场第1优先 0/3命中 vs 第4优先 3/3命中。
```

## 五、总结：我学到了什么

### 🔴 5个核心教训

**1. "教练很烂"是最不可靠的预测锚点**
Martínez 在 62' 做了一个世界杯淘汰赛史上最大胆的同时换人——四个球员一起上。81' 换下了 C罗。我们的框架说"他不会这么做"——他做了。教练可以在任何一场比赛中改变自己。**历史行为是教练能力的弱预测器，尤其在淘汰赛生死时刻。**

**2. "专家标签"（Livaković点球王、阿尔及利亚真实攻击力）在被挑战时会崩塌**
Livaković 面对 Ronaldo 的点球判断错误。阿尔及利亚的攻击力面对瑞士防线归零。**标签只有在不被挑战时才成立。淘汰赛就是挑战标签的地方。**

**3. 第4优先连续3场命中 → 框架的系统性偏差已被证实**
不是运气。框架给弱队进球分配了过多概率（§七过度触发），导致零封比分被压低。**"优先级别反转检查"应成为输出前的强制步骤。**

**4. 外部批评可能是正面信号，而非负面信号**
Schmeichel 批评 Martínez → 框架解读为"Martínez 更差"。实际：批评可能刺激了 Martínez 做出改变。**"反弹效应"应被纳入情境因子。**

**5. §七 需要内置对手防守级别折扣——不能等到赛后复盘才加**
连续3场 §七 误判（奥地利、阿尔及利亚以及即将验证的佛得角/加纳）→ 这不是修正案，这是框架的**结构性缺陷**。§七 在 v4 中是最被引用的规则，也是出错最多的规则。

### 🟢 3个验证成功的判断

**1. 🛡️保守策略的价值被再次验证**
7/3 复盘已经证明——保守（Dutching，不排除任何投注项）是唯一盈利的策略。如果两场都使用保守，葡萄牙 2-1葡 被保留（Kelly 排除了它），瑞士 2-0 也被保留。

**2. "知道弱点≠能执行"规则两次验证**
Petković 知道瑞士的弱点但没有武器执行。这是 7/3 瑞士报告中明确指出的——"知道瑞士的弱点是一回事，带着一个崩盘的门将群去执行是另一回事。"

**3. 晚间更新中"B席复出增加葡萄牙变阵可能性"的微弱信号**
在葡萄牙报告的晚间更新中，有一条被淹没的判断："B席复出可战——Martínez 多了一个变阵选择"。这是个微弱的正面信号——但被 §6.2 的压倒性"Martínez 不行"叙事压制了。**微弱的正确信号应当获得更多权重，尤其在它们与主流叙事相反时。**
