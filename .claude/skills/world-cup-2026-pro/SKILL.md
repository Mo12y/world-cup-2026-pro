# 零、v7 五大元规则（元规则 > 所有具体规则）

## 元规则 1：新规则必须通过反事实测试

```
新规则必须同时满足：
  □ 在至少 1 场过去正确的比赛上：加了规则 → 预测仍然正确
  □ 在至少 1 场过去错误的比赛上（非触发该规则的那场）：加了规则 → 预测被修正为正确
不通过 → 不进入框架。
```

## 元规则 2：新规则需要 3 场盲测期

```
新规则诞生 → 记录在"待验证规则"列表 → 观察 3 场 →
  3 场内判断被验证 ≥2 次 → 进入预测层
  3 场内无法验证或失败 → 删除
```

## 元规则 3：所有规则标注类型

```
[预测] — 赛前有输入数据，可独立产生判断
[解释] — 赛后或赛中才有数据，复盘用，不参与赛前预测
[争议] — 历史验证 < 3 次或有分歧
```

## 元规则 4：同一错误 ≥3 次才新增规则

```
1 次错误 → 记录
2 次错误 → 标注模式，等第 3 次
≥3 次错误 → 新增规则
```

## 元规则 5：双框架竞争

```
框架 A（基准线）：v3 精简版，~10 条规则，锁定不变
框架 B（进化版）：v7，持续迭代但需通过反事实+盲测

每轮新比赛：A 和 B 各自产出预测。赛后比较。
B > A 连续 3 轮 → B 规则合并入 A（A 升级）
B ≤ A → B 增量规则丢弃，回到 A
```

# 〇、赛前模式检查（强制 — 输出预测前必执行）

22 场中同一错误模式反复出现。这 5 个检查是框架的"免疫系统"。

```
□ 检查 1 — 体能/海拔/疲劳叙事？
    历史准确率：0/4（#17 摩洛哥 120'→#20 英格兰海拔→#22 比利时 120'，全错）
    如果是 → 降级为"弱信号"，不成为叙事锚点
    用 FRC 替代（恢复 ≥4 天 = 零折扣）

□ 检查 2 — §七 预测弱队面对 A 级防线进球？
    历史准确率：0/4（#11 奥地利/#13 阿尔及利亚/#17 加拿大/#21 葡萄牙，全零封）
    如果是 → A 级防线折扣 ×0.50（5/5 → 2.50 → 不自动预测进球）

□ 检查 3 — 核心叙事是否站在冷门方？
    历史：冷门叙事准确率 ~1/5
    如果是 → 强制执行 §5.5 Q3（"故事驱动数据？"）

□ 检查 4 — 本轮的锚点是否和上一轮犯同一个错？
    回看上一轮复盘的核心教训
    如果本轮分析的核心判断和上一轮的错误是同类型 → 标注"⚠️ 连续模式风险"

□ 检查 5 — 有没有一个 X 因素（球员/门将/教练换人）被框架忽略了？
    强制列出至少 1 个"框架可能不知道的事"
```

# 一、分析原则

## 1.1 因素权重排序

| 层级 | 类别 | 示例 | 权重 |
|------|------|------|:---:|
| 1 | **物理事实** | 海拔、伤病停赛、防线速度错位 | 最高 |
| 2 | **数据事实** | 连续不败/不胜、零封场次、攻防两端表现 | 中高 |
| 3 | **战术匹配度** | 阵型克制、关键对位、反击速度 vs 防线速度 | 中等 |
| 4 | **纸面标签** | 俱乐部名号、身价、豪门背景 | 低 |
| 5 | **历史叙事** | 魔咒、宿命、"X 年未赢淘汰赛" | 最低 |

🆕 **风格匹配史**：超过 2 次交手 + 每次有相同模式（如挪威 28 年 4 次对巴西进球）→ 升为第 3 层（战术匹配度）。

## 1.2 淘汰赛肥尾意识

```
├── 不排除小概率比分（<15%），保留在预测集底部
├── "应该发生的"在淘汰赛中经常不发生
└── 赛后区分「框架缺陷」和「随机方差」
```

# 二、攻击力评估 [预测]

## 2.1 攻击力预检

| # | 条件 | 含义 |
|---|------|------|
| ① | 小组赛 ≥2 场有进球 | 系统性破门 |
| ② | 队内有球员本届 ≥2 球 | 有验证过的终结者 |
| ③ | 小组赛攻破过 FIFA 前 15 对手 | 对强队验证过的火力 |
| ④ | 球队总进球 ≥5 | 火力充足 |
| ⑤ | 小组赛对 FIFA 前 15 对手拿分 | 竞技层级验证 |

```
分级判定：
  满足 ≥1 条 ①~④ → 真实攻击力，预测进球
  仅满足 ⑤ 且是有进球的平局 → 预测进球
  仅满足 ⑤ 且是 0-0 → 弱信号，走 §2.3
  0 条 → 走 §2.3
```

## 2.2 防守折扣（强制）[预测]

```
防线等级：
  A 级 — 场均失球 ≤0.5 且对 Top15 失球 ≤0.5 — 历史级防线（如西班牙 5 场 0 失球）
  B 级 — 场均失球 ≤1.0 — 良好防线
  C 级 — 场均失球 >1.0 或对 Top15 场均失球 ≥2.0 — 普通/有漏洞

折扣系数（🆕 v7 修正）：
  A 级 → ×0.50  ← 🔴 从 ×0.60 下调。4/4 历史：§七 面对 A 级从未进球
  B 级 → ×0.75
  C 级 → ×1.00（无折扣）

折扣后判定：
  ≥ 3.0 → 预测进球（概率 70-80%）
  2.0-2.9 → 边缘攻击力（40-50%），不自动预测
  < 2.0 → 不自动预测，走 §2.3
```

## 2.3 尊严进球审查 [预测]

预测弱势方进球必须基于可验证条件：

1. 强势方防线有**明确的可利用漏洞**
2. 弱势方中场能在高压下**实际向前锋输送球**
3. 弱势方锋线有**本届赛事实际兑现的终结数据**

```
淘汰赛阈值：≥1/3 → 预测进球
不满足 → 默认零封。
```

## 2.4 弱者韧性评估 [预测]

```
韧性信号：
├── 对 ≥2 支不同风格强队未被零封 → +20%
├── 有"落后→扳平"的场次 → +15%
├── 对前 5 球队拿分 → +15%
总分 ≥ 30% → 不默认零封，进球概率 ≥ 25%
```

## 2.5 大赛型门将折扣 [预测]

```
触发条件（≥2/3）：
  ① 前届世界杯有"单场救主"履历（≥3扑救零封 或 PK 扑≥2球）
  ② 本届场均扑救 ≥2.0
  ③ 对 Top 10 有零封记录（本届或上届）
触发 → 对方进球概率 ×0.85
```

## 2.6 核心缺失结构性折扣 [预测] ✅ 反事实通过

**反事实**：Hien（#6 瑞典）+ Koné（#17 加拿大）— 两次独立验证。

```
触发：确定缺席的核心中场枢纽 / 防线指挥官 / 进攻单核
折扣：本方攻击力 ×0.60 + 对手防线折扣降一级（A→B→C）
```

## FRC 体能恢复系数 [预测] ✅ 7/7 验证

**7/7 验证**：比利时 120 分钟 → FRC 1.0 → 零体能折扣 → 实际 9 分钟就进球。如果不用 FRC，v3 会犯错。

```
FRC = 基础恢复 × 深度系数 × 比赛强度

基础恢复：≥96h→1.0 / 72-96h→0.85 / <72h→0.70
深度：板凳≥4星→1.10 / 3星→1.00 / ≤2星→0.90
强度：120分钟→0.85 / 90分钟→1.00
FRC ≥ 1.0 → 无折扣
```

# 解释层（复盘专用 — 不参与赛前预测）

以下规则在复盘中有价值，但赛前缺乏输入数据、触发频率极低、或仅对单场比赛有效。赛前分析中**不使用**它们。

| 规则 | 移入解释层原因 |
|------|------|
| AAS 进球质量加权 | 输入数据在赛前难以获取；简化为只标记"注水"不触发额外折扣 |
| CTI 转化率倾向 | 需要 xG 数据（竞彩数据源不可用） |
| NDA 多核审查 | 仅加拿大场触发过一次 |
| §2.9 缺席类型 | 加拿大场专属区别 |

| HTA 半场条件修正 | 需半场 xG/射正数据（赛前不存在） |
| §4.1-§4.4 教练情境因子 | 事后解释教练行为，非事前预测；保留基线评级 |
| RSM 裁判风格匹配 | 20 场 0 次改变预测 |

检验：真实出场时间、国家队搭档场次、转会适应期、伤病恢复后状态。

# 五、比赛动态

## 5.1 点球路径 [预测]

```
冷门 ≥45% → 点球默认预测
冷门 35-45% → 点球备选，不替代 90 分钟预测
冷门 <35% → 点球不成为主要叙事
```

**比赛点球 ≠ 点球大战**：门将优势在点球大战中更显著。

## 5.2 冷门条件审查 [预测]

弱势方获胜需同时满足 3-5 个条件。最低实现路径 = 条件乘积 → 冷门概率基线。

## 5.3 优先级别反转 [预测]

```
触发：第 1 优先含弱队进球 + §2.2 折扣后弱队攻击力降级
操作：零封比分提升至第 1，标注"🔄 反转"
```

## 5.4 批判性质询 [强制]

```
Q1  核心叙事错了 → 哪个数据会第一个告诉我？
Q2  最脆弱的假设是什么？
Q3  我是否在用"故事"驱动结论？（自检：反转强弱关系后论据是否仍合理）
Q4  什么是框架"不可能知道"的？
Q5  上一轮同类比赛的实际结果？（应用到本轮 §2.2）
```

## 5.5 单队 3+ 球 [争议]

```
≥3/4 条件满足 → 单队 3+ 球概率 +12-15%
```

# 八、v7 执行步骤

**Step 1 — 信息搜集**

```
□ ① 官方大名单 — 核心球员确定缺席/存疑/首发
□ ② 准确的不败/连胜纪录（非二手汇总）
□ ③ 赛前 24h 最新伤病/阵容新闻
□ ④ 对方第二/第三攻击手评估（🆕 7/7 教训 — De Ketelaere）
□ ⑤ 上轮教训应用清单
```

**Step 2 — 分析（仅预测层规则）**

1. §2.1 攻击力预检 → §2.2 防守折扣（A 级 ×0.50）
2. §2.3 尊严进球 → §2.4 弱者韧性
3. §2.5 大赛型门将 → §2.6 核心缺失 → FRC
4. §3.1 崩塌检查 → §3.2 防守缺陷叠加
5. §4 教练基线 → §5.1 点球路径 → §5.2 冷门审查
6. §5.5 单队 3+ 球 → §七 不可量化因素

**Step 3 — 赛前模式检查**

执行 §〇 五个检查。全部通过才进入 Step 4。

**Step 4 — 批判性质询**

执行 §5.4 五项质询。标注最脆弱假设。

**Step 5 — 优先级别反转**

执行 §5.3。

**Step 6 — 输出**

- **实力差距 >2 档**：Top 6 比分排名 + 概率
- **实力差距 ≤2 档**（🆕）：方向 + 总进球数区间 + BTTS 概率。不输出精确比分排名。
  - 原因：精确比分在实力接近比赛中命中率 ~15%，22 场数据

**Step 7 — 赛后验证**

对比预测 vs 实际 → 追溯偏差到具体规则 → 区分框架缺陷 vs 随机方差 → 记录偏差日志。
