# v8 决策过滤器

## 为什么 v1-v7 被废弃

24 场数据，一个残酷的事实：

| 系统 | 1/8 决赛晋级方 | 规则数 |
|------|:---:|:---:|
| FIFA 排名裸猜 | 6/8 (75%) | 0 |
| v7 量化框架 | 5/8 (62.5%) | 15 |
| **v8 决策过滤器** | **7/8 (87.5%)** | **3 条推翻条件** |

v7 的量化分析——AAS、CTI、双维度防线、大赛型门将、核心缺失折扣、NDA、USQ、HTA、FRC、RSM、教练情境因子——在 1/8 决赛中产生了**负净贡献**。分析越用力，预测越偏。

v7 比裸猜多错的两场（墨西哥、葡萄牙），原因完全相同：**一个"好故事"说服框架主动推翻了正确的默认判断。** 海拔叙事 → 墨西哥不败。伊比利亚德比叙事 → 双方都会进球。两个故事都被比赛打脸。

**v8 不做分析。v8 做决策。**

## 核心规则

```
默认：FIFA 排名更高的球队晋级。

推翻默认 — 需 ≥2 条同时触发：
  □ ① 排名更高方有核心球员确定缺席
       → 核心 = 中场攻防枢纽 / 防线指挥官 / 进攻组织单一枢纽
       → 非核心（边后卫/边锋/轮换球员/门将）不触发
  □ ② 排名较低方有 A 级防线
       → 淘汰赛阶段 ≤1 场丢球 且 对 FIFA Top 15 有零封记录
  □ ③ 排名较低方有顶级终结者 + 排名更高方防线有明确漏洞
       → 顶级终结者 = 本届赛事 ≥5 球
       → 防线漏洞 = 淘汰赛阶段连续 ≥2 场丢球

不满足 ≥2 条 → 维持默认。不分析。不叙事。不调整权重。
```

## 执行步骤

**Step 1 — 查 FIFA 排名**：确定默认晋级方。

**Step 2 — 检查 3 条推翻条件**：
- ① 排名更高方有核心缺席？
- ② 排名较低方有 A 级防线？
- ③ 排名较低方有 ≥5 球终结者 + 排名更高方淘汰赛连续丢球？

**Step 3 — 计数**：
- ≥2 条触发 → 推翻默认，排名较低方晋级
- 0-1 条触发 → 维持默认

**Step 4 — 比分和总进球估算**（简化）：
```
默认方 A 级防线 → 低比分（0-1, 0-2, 1-1）
默认方 B/C 级防线 + 劣势方有 ≥5 球终结者 → 双方进球
双方 A vs A 级防线 → 极低比分（0-0, 1-0, 0-1），点球路径可能
双方 B/C 级 → 正常比分（1-2, 2-1, 2-2）
```

**Step 5 — 输出**：
- 晋级方预测
- 核心逻辑（默认 or 推翻，哪几条触发）
- 比分区间 + BTTS 概率
- ⚠️ 置信度标签

# 资金管理铁律（投注策略强制执行）

> 来自 24 场实盘验证：投注亏损的主因不是预测不准，是亏损后加大赌注。

```
单日投注上限 = 累计盈利 × 系数
  累计盈利 > 0   → 上限 = 累计盈利 × 2（最小 20 元）
  累计盈利 ≤ 0   → 上限 = 20 元

单方案上限 = 单日上限 × 50%
锚点分散 = 任何两个方案不得共享同一个致命锚点
```

**7/8 教训**：累计 +11 → 单日投 94 → 跌到 -83。94 元是上限的 8.5 倍。以后由投注策略强制执行上限。
