```

净利润   = 命中回报 - 总投入
```

# 📐 Kelly 模式

## 投注范围

默认仅使用胜平负和比分两种投注方式。让球胜平负、总进球数、半全场不在默认范围内，仅当用户明确要求时纳入。

```

【分析报告预测结果】

```

> 每场投入金额必填，未提供则追问。

### 胜平负自动推导

```
全部预测比分都是主胜比分 → 推导胜平负 = 主胜
全部预测比分都是平局比分 → 推导胜平负 = 平
全部预测比分都是主负比分 → 推导胜平负 = 主负
混合（如同时包含 1:0 和 1:1）→ 追问用户
```

推导出的胜平负自动加入投注池。

## 三策略

| 策略 | 分配逻辑 | 角色 |
|------|----------|------|
| 🔥 激进 | Kelly λ=1.0 | 对数财富增长最优 |
| ⚖️ 折中 | EV 排序均分 | 忽略方差，只看预期回报 |
| 🛡️ 保守 | 风险约束 ≤N/3 + Dutching | 最差净利 ≥ -N/3 |

### 🔥 激进 — Kelly λ=1.0

```
kᵢ = pᵢ - (1-pᵢ)/(bᵢ-1)
kᵢ' = max(kᵢ, 0)          ← kᵢ ≤ 0 则排除
wᵢ = kᵢ' / Σ kⱼ'
金额ᵢ = round(N × wᵢ / 2) × 2
```

### ⚖️ 折中 — EV 排序均分

```
EVᵢ = pᵢ × bᵢ - 1
EV ≤ 0 → 排除
EV > 0 → 按 EV 降序，N 均分给所有正 EV 项
金额 = round(N / 正EV项数 / 2) × 2
```

无正 EV → 全部均匀分配 N（fallback）。

### 🛡️ 保守 — 风险约束 Dutching

```
总下注上限 = N / 3（精确到偶数）
权重 ∝ 1/赔率ᵢ
金额 = round(上限 × 权重 / 2) × 2
保留 2N/3 作为现金缓冲
```

包含所有投注项（不排除负 EV），最差净利 ≥ -N/3。

### EV 计算（有概率时）

```
EV = (Σ pⱼ × 赛果j净利润) + (1 - Σ pⱼ) × (-N)
🛡️保守：现金缓冲（2N/3）视为 0 回报，仅在全部错误时缓冲亏损
```

### 无概率 fallback

所有策略统一：均匀分配 N / 投注项数。

### 叠加命中检测

检查预测比分是否也属于胜平负投注的结果类型，标注叠加命中总回报。

### 差异度检测

- 三策略金额分布是否都不同？→ 「三策略分化充分」
- 某两策略金额完全一样？→ 「X 与 Y 趋同」
- 所有金额差异 < 5 元 → 「N 太小导致舍入掩盖差异」

## 二串一（两种组合，各 10 元）

```
类型一：最优项二串一 → 每场 Kelly 权重最高项组合
类型二：大方向二串一 → 每场胜平负大方向组合

二串一赔率 = 赔率₁ × 赔率₂
投入 = 10 元（5 注）
命中回报 = 10 × 赔率
净利润 = 命中回报 - 10
总投入 = 20 元
```
