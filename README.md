# world-cup-2026-pro

2026 世界杯预测分析项目。24 场实盘验证，8 轮迭代（v1→v8）。

**核心发现：量化分析在淘汰赛中没有产生正价值。v8 用 3 条推翻条件的决策过滤器替代了 v1-v7 的 30 条量化规则，1/8 决赛回测 87.5% 晋级方准确率。**

> ⚠️ **研究项目**。不构成投注建议。

## v8 决策过滤器

```
默认：FIFA 排名更高方晋级。

推翻默认 — 需 ≥2 条同时触发：
  □ ① 排名更高方有核心球员确定缺席
  □ ② 排名较低方有 A 级防线（淘汰赛 ≤1 场丢球 + 对 Top 15 零封）
  □ ③ 排名较低方有 ≥5 球终结者 + 排名更高方淘汰赛连续丢球

不满足 → 维持默认。不叙事。不调整权重。
```

## 为什么 v1-v7 被废弃

v7 的量化框架在 1/8 决赛中比"什么都不做"更差。两次方向错误（墨西哥、葡萄牙）都是同一个模式：**一个"好故事"说服框架主动推翻了正确的默认**——海拔叙事 → 墨西哥不败，伊比利亚德比叙事 → 双方进球。

8 轮迭代的核心教训：

| 发现 | 证据 |
|------|------|
| 规则越多越不准 | v3(10条) 100% → v6.1(30条) 0% |
| 复盘→投注有裂缝 | 7/4 写了"方向>比分"→7/5 全押比分 |
| 叙事绑架反复出现 | 体能/海拔/克星 — 5次，全错 |
| 亏损主因不是预测 | 是亏损后加大赌注（12→94元） |

## 资金管理铁律

来自 24 场实盘：亏损主因不是预测不准，是亏损后加大赌注。

```
单日上限 = 累计盈利 > 0 ? 累计盈利 × 2 : 20 元
单方案上限 = 单日上限 × 50%
```
