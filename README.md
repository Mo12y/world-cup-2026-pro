# world-cup-2026-pro

2026 美加墨世界杯分析预测框架。经过 16 场 1/16 决赛实盘验证、6 轮迭代（v1→v6），构建了 30 个量化组件的分析引擎，并配套独立的竞彩投注策略。

> ⚠️ **Pre-alpha 研究项目**。不构成投注建议。仅供学习和研究使用。

## 核心能力

### 30 组件量化分析框架（16 场验证）

| 模块 | 组件 |
|------|------|
| 攻击力 | 预检(5条件) → 防守折扣(A/B/C级 ×0.60/0.75/1.00) → 尊严进球 → 弱者韧性 → 早期进球 |
| 防守 | 结构性崩塌连锁 → 缺陷叠加互爆 → 纸面vs实际可执行 |
| 教练 | 基线+情境因子(±0.5档) → 战术内幕武器匹配(乘积) → 领先失误 → 反弹效应 |
| 动态 | 纪录质量调整 → 点球路径(45%门槛·比赛vs大战区分) → 反转检查 → 批判性质询(5问) |
| 裁判 | 定性标注，不修改概率 |

### 16 场 1/16 决赛验证结果

| 指标 | 成绩 |
|------|:---:|
| 晋级方预测 | **14/16 (87.5%)** |
| 90' 比分完全命中 | 7/16 (43.8%) |
| 比分在预测集内 | 14/16 (87.5%) |
| 前 7 场比分完全命中 | 7/7 (100%) |

### 竞彩投注策略

三策略框架（🔥激进 Kelly λ=1.0 / ⚖️折中 EV 排序均分 / 🛡️保守 ≤N/3 Dutching）+ 二串一。
保守 Dutching 为 16 场实盘验证中唯一连续盈利策略（7/3 +31.9% ROI）。

## 用法

### 前提条件

- [Claude Code](https://claude.ai/code) CLI 或 IDE 扩展
- 项目安装为 Claude Code skill

### 分析比赛

```
预测 巴西 vs 挪威
分析 葡萄牙 vs 西班牙
阿根廷能晋级吗
```

### 生成投注策略

```
帮我规划投注 [附带预测概率+赔率]
算一下投注 [附带投注项]
```

### 赛后复盘

```
验证一下 [附带实际比分]
帮我复盘今天的比赛
```
